# 方案

## 用户偏好

- 排版引擎：LaTeX (xelatex)
- 竞赛类型：电工杯（兼容国赛/美赛）
- 论文语言：中文
- 子问题数量：4个（Q1车辆配送、Q2车+无人机、Q3容量变化、Q4选址+路径）
- 正确答案参考：Q1最短路径约582km，配送时间约6.32h

## workflow

| step | skills | 作用 |
|------|--------|------|
| 1. 赛题分析与建模设计 | `2analysis-modeling` | 读取题面和数据，拆解4问，建数学模型 |
| 2. 编程实现和图表生成 | `3coding-visual` | 写求解代码，跑结果，生成图表 |
| 3. 流程与架构图绘制 | `4drawio` | 画技术路线图、算法流程图 |
| 4. 竞赛论文撰写 | `5writing` | 基于真实结果写LaTeX论文 |
| 5. 验证和验收 | `6verity` | 检查数值一致性、图表完整性 |
